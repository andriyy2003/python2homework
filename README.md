Це мій навчальний проєкт, виконаний у PyCharm.

🚀 Виконане завдання
Створення власної функції для виведення вітання.

🛠 Що я використав:
def — для створення (визначення) власної функції.

print() — для виведення тексту "Привіт, світ!" на екран.

Виклик функції — написання назви функції з дужками для її запуску.

💻 Як працює код:
Програма визначає функцію, яка містить у собі команду виводу тексту. Після цього ми викликаємо цю функцію за ім'ям, і вона миттєво виконує збережену в ній дію.
